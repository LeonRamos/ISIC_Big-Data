Instalación y Configuración del Entorno de Desarrollo para Big Data

###  Propósito de la actividad

Configurar el entorno de trabajo que se utilizará **durante todo el semestre** para el desarrollo de prácticas relacionadas con **Big Data, análisis de datos e ingeniería de datos**, empleando herramientas profesionales ampliamente utilizadas en la industria.

Todas las instalaciones deberán realizarse desde sus **páginas oficiales**, **sin usar Anaconda u otros paquetes preconfigurados**, y utilizando la **cuenta institucional** para acceder a licencias y beneficios académicos.

---

## Requisitos generales

* Computadora con **Windows 11**
* Conexión estable a Internet
* Cuenta de correo **institucional activa**
* Permisos de administrador en el equipo

---

## Herramientas a instalar

Las siguientes herramientas serán obligatorias:

1. **Lenguaje R**
2. **Python**
3. **Docker Desktop**
4. **PostgreSQL**
5. **Visual Studio Code (VS Code)**

---

## Parte 1: Instalación de Python (desde sitio oficial)

1. Accede a la página oficial:

   * [https://www.python.org](https://www.python.org)

2. Descarga la versión:

   * **Python 3.x (64 bits) para Windows**

3. Ejecuta el instalador y:

   * Marca la opción **“Add Python to PATH”**
   * Selecciona **Install Now**

4. Verifica la instalación:

   * Abre *Símbolo del sistema* y ejecuta:

     ```
     python --version
     ```

---

## Parte 2: Instalación del Lenguaje R

1. Accede al sitio oficial:

   * [https://cran.r-project.org](https://cran.r-project.org)

2. Selecciona:

   * *Download R for Windows*
   * *Base → Download R*

3. Instala con las opciones por defecto.

4. Verifica abriendo **R GUI** desde el menú inicio.

---

## Parte 3: Instalación de Visual Studio Code (VS Code)

1. Accede a:

   * [https://code.visualstudio.com](https://code.visualstudio.com)

2. Descarga **VS Code para Windows (64 bits)**

3. Instala y abre el programa.

4. Instala las siguientes extensiones desde VS Code:

   * **Python** (Microsoft)
   * **R** (REditorSupport)
   * **Docker** (Microsoft)
   * **PostgreSQL** (opcional)

---

## Parte 4: Instalación de PostgreSQL

1. Accede a la página oficial:

   * [https://www.postgresql.org](https://www.postgresql.org)

2. Descarga:

   * **PostgreSQL para Windows**

3. Durante la instalación:

   * Define una contraseña segura para el usuario `postgres`
   * Acepta la instalación de **pgAdmin**

4. Verifica ingresando a **pgAdmin 4** y conectándote al servidor local.

---

## Parte 5: Instalación de Docker Desktop

1. Accede al sitio oficial:

   * [https://www.docker.com/products/docker-desktop/](https://www.docker.com/products/docker-desktop/)

2. Crea una cuenta utilizando tu **correo institucional**.

3. Solicita y activa **Docker Student License** (plan educativo).

4. Descarga e instala **Docker Desktop para Windows**.

5. Verifica ejecutando:

   ```
   docker --version
   ```

---

## Parte 6: Registro con cuenta institucional y beneficios académicos

El estudiante deberá:

* Usar **exclusivamente su correo institucional** para:

  * Docker (licencia educativa)
  * GitHub Student Developer Pack (recomendado)
  * Herramientas académicas vinculadas a VS Code


---

## Parte 7: Verificación del entorno

El estudiante deberá comprobar:

* Python ejecuta desde terminal
* R abre correctamente
* VS Code reconoce Python y R
* PostgreSQL está activo y accesible desde pgAdmin
* Docker Desktop inicia sin errores
